# Data-analysis
Analisi dati per esperimenti di fisica. 
Durante il laboratorio di fisica è stato utilizzato il codice per effettuare l'analisi dati, ricavando con varie tecniche statistiche tra cui il metodo dei minimi quadrati, il best fit, la massima verosimiglianza, i parametri delle distribuzioni che si sono poi rivelati quantità fisiche ricercate durante l'esperimento. 
Tra le librerie sono state utilizzate:
-Numpy per la gestione degli array contenenti i dati;
-Scipy per la statistica;
-Matplotlib per la visualizzazione dei grafici.

Data analysis for physics experiments
During the physics laboratory, a custom codebase was developed to perform data analysis, utilizing various statistical techniques—including Ordinary Least Squares (OLS), Best Fit optimization, and Maximum Likelihood Estimation (MLE)—to derive distribution parameters that represented the physical quantities sought during the experiment. The following libraries were core to the project:
-Numpy for management and manipulation of the arrays;
-Scipy for implementing advanced statistical models and curve fitting;
-Matplotlib for data visualization through plots;
